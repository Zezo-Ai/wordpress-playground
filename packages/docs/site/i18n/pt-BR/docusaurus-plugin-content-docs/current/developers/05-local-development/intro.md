## Desenvolvimento Local do WordPress com Playground

<!-- Playground offers various development environments to streamline setting up and managing WordPress sites. -->

O Playground oferece vários ambientes de desenvolvimento para simplificar a configuração e gerenciamento de sites WordPress.

<!-- For a quick start, use a [public Playground web instance](/web-instance) at https://playground.wordpress.net/. Alternatively, you can [host your own WordPress Playground](/developers/architecture/host-your-own-playground). -->

Para um início rápido, use uma [instância web pública do Playground](/web-instance) em https://playground.wordpress.net/. Alternativamente, você pode [hospedar seu próprio WordPress Playground](/developers/architecture/host-your-own-playground).

<!-- Playground also provides tools for local WordPress development, prioritizing easy installation and usability: -->

O Playground também fornece ferramentas para desenvolvimento local do WordPress, priorizando instalação fácil e usabilidade:

<!-- -   **[@wp-playground/cli](/developers/local-development/wp-playground-cli):** A command-line tool to quickly launch a WordPress site from your terminal. -->
<!-- -   **[Visual Studio Code Extension](/developers/local-development/vscode-extension):** Integrates WordPress development directly into VS Code for a seamless experience within the popular editor. -->

-   **[@wp-playground/cli](/developers/local-development/wp-playground-cli):** Uma ferramenta de linha de comando para lançar rapidamente um site WordPress do seu terminal.
-   **[Extensão do Visual Studio Code](/developers/local-development/vscode-extension):** Integra o desenvolvimento WordPress diretamente no VS Code para uma experiência perfeita dentro do editor popular.

<!-- For those needing more control, Playground offers tools for Node.js: -->

Para aqueles que precisam de mais controle, o Playground oferece ferramentas para Node.js:

<!-- -   **[WordPress Playground in Node.js](/developers/local-development/php-wasm-node):** For low-level control over the WebAssembly PHP build, explore the [`@php-wasm/node` package](https://npmjs.org/@php-wasm/node), which includes the PHP WebAssembly runtime. -->

-   **[WordPress Playground no Node.js](/developers/local-development/php-wasm-node):** Para controle de baixo nível sobre a compilação WebAssembly do PHP, explore o pacote [`@php-wasm/node`](https://npmjs.org/@php-wasm/node), que inclui o runtime WebAssembly do PHP.
